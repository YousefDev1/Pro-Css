## Installation

Install via git `git clone https://github.com/YousefDev1/Pro-Css`

```text
pro/
├── css/
│   ├── pro.css
│   └── icons
        └── pro-icons.css
└── js/
    └── pro.js
```

## Get Start

You can start by adding framework files to your project.
**Css files**
    <br>
    `<link rel="stylesheet" href="pro/css/pro.css">`
    
**JavaScript files**
    <br>
    `<script src="pro/js/pro.js"/script>`
